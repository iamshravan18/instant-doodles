# Independent InstaDoodle guide

A static-first Next.js independent guide to InstaDoodle’s AI whiteboard animation workflow. It includes environment-safe canonicals, robots and sitemap routes, truthful WebPage/Breadcrumb JSON-LD, accessible navigation, local official product visuals, and no unverified commercial claims.

## Commands

```bash
npm run dev
npm run lint
npm run build
```

Set `NEXT_PUBLIC_SITE_URL` only for the independent production domain. Without it, pages remain noindex, robots disallows crawling and the sitemap remains empty. Add pricing, guarantee, review, checkout, company and legal data only after business verification.
